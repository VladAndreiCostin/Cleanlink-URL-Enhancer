Frequently Asked Questions

Everything you need to know about link cleaning and privacy.

How do I remove fbclid and gclid from URLs?

Cleanlink URL Enhancer acts as an automatic URL cleaner. When you copy a link containing tracking IDs like Facebook's fbclid or Google's gclid, the extension instantly intercepts the clipboard data and strips those parameters before you paste it.

Can I keep my own UTM parameters while stripping others?

Yes. Unlike basic clean link extensions that wipe all query strings indiscriminately, Cleanlink URL Enhancer allows you to set custom presets. It will scrub the target URL of foreign tracking data, and then automatically append your specified UTM parameters or affiliate tags.

Is this extension safe? Where does my data go?

Cleanlink URL Enhancer is a 100% local, privacy-first utility. We use zero cloud servers and do not run behavior analytics. The extension requires zero permissions beyond "read click data," meaning the URL processing happens entirely on your own machine.
